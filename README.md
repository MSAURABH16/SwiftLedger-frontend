# 🏦 SwiftLedger

A modern banking dashboard built with React and Vite that provides a clean and intuitive interface for managing accounts, monitoring balances, and transferring funds securely.

## 🌐 Live Demo

🔗 **Live Application:** https://swiftledger.vercel.app

### 🔐 Authentication
- User Registration

- Secure Authentication
- Protected Routes
- Logout Functionality

### 💳 Account Management
- Create New Accounts

- Account Overview Dashboard

### 💸 Money Transfers
- Transfer Funds Between Accounts
- Transaction Validation

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/swiftledger-frontend.git
cd swiftledger-frontend
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_API_URL=your_api_url
```

### Start Development Server

```bash
npm run dev
```

The application will be available at:

```text
https://swiftledger.vercel.app
```
